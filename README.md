# لاند إكس - منصة استثمارات الأراضي الموسمية

منصة وطنية لاستثمارات الأراضي الموسمية، تهدف إلى تمكين المستثمرين من الوصول إلى فرص استثمارية آمنة وموثوقة في القطاع الزراعي في المملكة العربية السعودية.

## المشروع

منصة SaaS للتعاون التدريبي/تخرج تمكن من:
- نشر فرص استثمارية موسمية من قبل البلديات
- تصفح الفرص من قبل المستثمرين
- إدارة شاملة للمنصة من قبل الإدارة

## التقنيات

### Frontend (المرحلة 1)
- React 19
- Vite
- Tailwind CSS
- React Router

### Backend (المرحلة 2 - مخطط)
- Supabase (الموصى به) أو Flask + PostgreSQL

## البنية المعمارية

### الأدوار
1. **الزائر العام** - تصفح الفرص والأخبار
2. **البلدية/الجهة الحكومية** - إدارة الفرص والإعلانات
3. **إدارة المنصة** - إدارة المنصة والمستخدمين

### فصل المسؤوليات
- **إدارة المنصة**: العمليات التشغيلية للمنصة
- **البلديات**: إدارة المحتوى التنظيمي للفرص الاستثمارية

## التثبيت

```bash
# تثبيت التبعيات
npm install

# تشغيل خادم التطوير
npm run dev

# بناء للإنتاج
npm run build
```

## الصفحات

### الصفحات العامة
- `/` - الصفحة الرئيسية
- `/opportunities` - الفرص الاستثمارية
- `/opportunities/:id` - تفاصيل الفرصة
- `/about` - عن المنصة
- `/contact` - تواصل معنا
- `/news` - الأخبار

### المصادقة
- `/login` - تسجيل الدخول
- `/register` - إنشاء حساب

### لوحات التحكم
- `/investor/dashboard` - لوحة المستثمر
- `/municipality/dashboard` - لوحة البلدية
- `/admin/dashboard` - لوحة الإدارة
